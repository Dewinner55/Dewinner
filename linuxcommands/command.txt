cltr+alt+T Команда для открытия терминала
 ~ -> символ домашней директории
 / -> символ перехода с одной директории в другую, еще и обозначает корневую директорию
 touch (создаёт указанный файл (пример тектоствый документ))
 .. -> обозначает предыдущий директорию
 . -> обозначает текущую директорию
 mkdir jonh snow (создать папку(директорию) mkdir (make directory) - команда для создания директории
 rmdir jonh snow (удалить папку(директорию))(Невозможно удалить если в папке есть файлы)(удаляет пустые директории)
 rm -rf(принудительная удаление все что есть внутри этой папки(директории))
 rm удаляет файлы
 pwd(print working directory) путь до текщей папки(директории)
 mv команда для перемещение(mv jonh.txt (путь перемещение) ~/desktop)  и переименновывание(mv jonh.txt hello.txt)
 history (история команд)
 ls -a -> команда для вывода всех файлов и директорий в текущей директории
 cat <filename> - выводит в терминал все содержимое файла
 echo - команда для быстрой записи тествой информации в какой либо файл echo 'текст' > название_файла
 Nano <path_to_file> - открывет файл для редактирования в графическом интерфейсе *Если нет такого файла то nano создаст новый файл
 sudo apt install установка программ
